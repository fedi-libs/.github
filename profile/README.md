# About fedi-libs
fedi-libs is created with the goal of expanding Python libraries related to the Fediverse.

fedi-libs now focus for maintenance ActivityPub library (e.g, apkit, apmodel, apsig)

## Libraries
- [apkit](https://github.com/fedi-libs/apkit) is a modern, fast, and powerful toolkit for building ActivityPub-based applications with Python, based on standard Python type hints.
- [apmodel](https://github.com/fedi-libs/apmodel) is ActivityStreams 2.0 and nodeinfo 2.0/2.1 implementation for Python.
- [apsig](https://github.com/fedi-libs/apsig) is implemention of draft-cavage-http-signatures-12, rsasignature2017 (ld signature) and fep-8b32 (Object Integrity Proofs).

## Server
- [kagura](https://github.com/fedi-libs/kagura) is mastodon api-compatible activitypub implemention, based on apkit.
